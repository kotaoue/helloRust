# egui_hello

[egui](https://github.com/emilk/egui) フレームワークを使ったシンプルな Hello World GUI アプリです。

## 実行方法

`egui_hello` ディレクトリに移動して `cargo run` を実行します。

```bash
cd egui_hello
cargo run
```

ウィンドウが開き、**Hello, World!** と表示されます。
