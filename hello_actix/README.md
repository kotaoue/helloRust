# hello_actix

This is a simple Hello World web application built with the [Actix Web](https://actix.rs/) framework.

## Usage

```sh
cd hello_actix
cargo run
```

Once the server is running, open your browser or use `curl` to access it:

```sh
curl http://127.0.0.1:8080/
```

```sh
curl http://127.0.0.1:8080/sing
```

```sh
curl http://127.0.0.1:8080/sing
```
