# hello_egui

This is a simple Hello World GUI application built with the [egui](https://github.com/emilk/egui) framework.

## Features

* **Pure Rust** — No C/C++ dependencies; implemented entirely in Rust
* **Immediate mode** — UI is rebuilt every frame, making state management simple and predictable
* **Cross-platform** — Runs on Linux, macOS, Windows, and WebAssembly (WASM)
* **Lightweight & fast** — Minimal dependencies and fast compile times
* **Highly customizable** — Flexible theming, fonts, and layouts
* **eframe integration** — The `eframe` crate lets a single codebase target both native desktop and web

## Usage

```sh
cd hello_egui
cargo run
Hello, World!
```
